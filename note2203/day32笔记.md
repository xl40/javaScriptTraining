# 二、jq的使用 - 数据请求
1. $.get(url, data, success, resDataType)
    - 专门用来发起get请求
    - 类似于咱们的ajaxGet
2. $.post(url, data, success, resDataType)
    - 专门用来发起post请求
    - 类似于咱们的ajaxPost
3. $().load()
    - 专门用来发起html文件请求
4. $.ajax()
    - 可以发起所有请求，包括jsonp
    - 类似于咱们的数据请求三合一
